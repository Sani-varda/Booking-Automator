# MedSpa Booking Automator

A production-grade micro-SaaS for missed-call-to-booking automation.

## Features
- Missed Call SMS Auto-Response
- AI Triage Agent
- Booking Integration (Calendly/GHL)
- Lead Dashboard

## Deployment
This project follows the "Deploy Your Own SaaS" principles. Use Docker Compose for a quick setup.

```bash
docker-compose up --build
```
